j4status

A plugin-based status line generator

j4status provides a status line using several plugin to retrieve information from your computer.

j4status is a part of the j4tools tools set.

The j4status-plugins project provides several additional plugins to j4status.

Dependencies

  • GLib
  • xsltproc and DocBook man page XSL stylesheets (for man pages generation)
  • For the i3bar output plugin:
    • yajl (2.0.0 or later)
  • For the Netlink input plugin:
    • libl (3 or later)
  • For the Upower input plugin:
    • libupower-glib
  • For the sensors input plugin:
    • libsensors from lm-sensors
  • For the PulseAudio plugin:
    • libpulse (from PulseAudio)
    • libpulse-mainloop-glib (from PulseAudio)
  • For the MPD input plugin:
    • libmpdclient
